Accessibiliy Issues

  • Only tab keyboard support
    • Menu button pattern requires the links to act like a menu, which included supporting the cursor keys.
    • Menu button pattern requires the tab key to close the menu and move to next focusable item on page.
    • If only tab key is support, the disclosure pattern should be used instead of the menu button pattern.
  • No description of behaviors to assistive technologies.
  • No keyboard focus styling.
