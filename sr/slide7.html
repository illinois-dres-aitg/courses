JAWS: Keyboard Shortcuts

Windows Desktop/Applications

Key Description
Control+ESCape Start menu
Alt+Tab Move between windows
Alt+F4 Close window
F10 Main menu

Internet Explorer/Edge Keyboard Keyboard Shortcuts (same at Firefox)

Key Description
Control+L URL address bar
F6 Move focus between web content and sidebar
Control+T Open new tab
Control+Tab Move to next tab
Control+F4 Close tab

Basic JAWS Keyboard Shortcuts

Key Description
JAWS Key Insert key
Insert+J Open NVDA Preferences
Insert+J, Alt+F4 Moves focus to JAWS applicatin window and then exitx JAWS using Alt+F4

JAWS Keyboard Shortcuts for Web Browsing

Key Description
Tab Next link
Shift+Tab Prior link
Insert+F7 List of links dialog box
L Next list
Shift+L Prior list
I Next item in a list
Shift+I Previous item in a list

JAWS Keyboard Shortcuts for Searching Text Content

Key Description
Control+F Pops up a dialog in which you can type some text to find in the current document
F3 Finds the next occurrence of the text in the document that you previously searched for
Shift+F3 Finds the previous occurrence of the text in the document you previously searched for
